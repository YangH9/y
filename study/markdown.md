#### 标题

> 有6种标题变体。符号 "#" 的数量，以及后面的文字，表明标题的重要性。

``` md
# 标题1
## 标题2
### 标题3
#### 标题4
##### 标题5
###### 标题6
```

## 标题2

#### 强调

> 修改文本是如此整洁和容易。你可以让你的文字变成粗体、斜体、删除线、高亮、下标或上标。

``` md
**粗体**
__粗体__
*斜体*
_斜体_
**粗体_倾斜_**
~~删除线~~
下~标~
上^标^
高==亮==
```

**粗体**
__粗体__
*斜体*
_斜体_
**粗体_倾斜_**
~~删除线~~

上^标^
