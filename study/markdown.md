#### 标题

> 有6种标题变体。符号 "#" 的数量，以及后面的文字，表明标题的重要性。

``` md
# 标题1
## 标题2
### 标题3
#### 标题4
##### 标题5
###### 标题6
```

# 标题1
## 标题2
### 标题3
#### 标题4
##### 标题5
###### 标题6

#### 强调

> 修改文本是如此整洁和容易。你可以让你的文字变成粗体、斜体、删除线、高亮、下标或上标。

代码 | 预览
| :-: | :-:
``` **粗体** ``` | **粗体**
``` __粗体__ ``` | __粗体__
``` *斜体* ``` | *斜体*
``` _斜体_ ``` | _斜体_
``` **粗体_倾斜_** ``` | **粗体 _倾斜_**
``` ~~删除线~~ ``` | ~~删除线~~
``` <u>下划线</u> ``` | <u>下划线</u>
``` <small>小号字体</small> ``` | <small>小号字体</small>
``` <big>大号字体</big> ``` | <big>大号字体</big>
``` 文本^上标^ ``` | 文本^上标^
``` 文本~下标~ ``` | 文本~下标~
``` <mark>文本高亮<mark> ``` | <mark>文本高亮<mark>
